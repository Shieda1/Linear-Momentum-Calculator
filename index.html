What is the linear momentum?
Momentum is the product of the mass and velocity of an object. It is a vector quantity and it possess a magnitude and a direction.
How to calculate linear momentum?
To calculate linear momentum you should use the following formula:
p = m * v
where:
p = momentum
m = mass
v = velocity
Why use linear momentum?
Linear momentum is useful in cases where there is a few objects which interact with each other but not with the rest of the world.
